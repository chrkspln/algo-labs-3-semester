**Програма 2_1.**

Програма визначає, чи є введене число парним.
```c
#include <stdio.h> 
#include <stdlib.h> 
#include <iostream> 
int main()
{
int n;
puts("Input number:"); 
scanf("%d",&n);
if(n%2==0) 
	printf("number %d even \n", n); 
else 
	printf("number %d odd\n\n", n); 
system("pause");
return 0;
}
```
